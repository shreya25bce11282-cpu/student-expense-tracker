# 💸 Expense Tracker

A modern, mobile first expense tracking application built as part of **Design2Code 2.0 - Round 2**. The project transforms an assigned wireframe into a fully functional frontend experience, focusing on accuracy, responsiveness, usability, and clean implementation.

## 📖 Overview

Managing personal finances shouldn't feel like solving a puzzle. This Expense Tracker provides an intuitive interface for monitoring spending habits, setting budgets, tracking savings goals, and gaining insights through visual analytics.

The application follows the provided Design2Code wireframe while implementing interactive features that enhance the overall user experience.

---

## ✨ Features

### 🏠 Dashboard

* Overview of total budget and remaining balance
* Savings progress tracking
* Financial health score
* Smart spending insights
* Recent transactions section

### 💰 Expense Management

* Add new expenses
* Edit existing expenses
* Delete expenses
* Categorized expense tracking
* Expense history view

### 📊 Analytics & Reports

* Category wise spending breakdown
* Interactive donut chart visualization
* Spending trend analysis
* Top spending category detection
* Weekly, Monthly, and Yearly report views

### 🎯 Budget Planning

* Set monthly budget
* Configure category wise limits
* Budget utilization indicators
* Overspending warnings

### 🔔 Notifications

* Budget alerts
* Spending reminders
* Activity notifications

### 👤 Profile Management

* User profile information
* Financial preferences
* Goal tracking

### 📱 Responsive Design

* Mobile-first interface
* Smooth navigation experience
* Modern glassmorphism-inspired UI
* Animated counters and transitions

## 🚀 Installation & Setup

### Clone the Repository

```bash
git clone <repository-url>
cd expense-tracker
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
